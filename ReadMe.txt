To run the program first install django using 

    pip install django

or if you are using an anaconda enviroment

    conda install -c anaconda django

To install bootstrap use the command 

    pip install django-bootstrap-v5

or if you are using an anaconda enviroment use the command

    conda install -c conda-forge bootstrap

Next use the terminal to navigate to the root directory of the project (CS396) and use the command

    python manage.py runserver

Now navigate to 127.0.0.1:8000 in your web browser to begin using the program
